# Mubashir Hassan Portfolio

Static GitHub Pages portfolio for `MubashirHassan1436.github.io`.

## Structure

- `index.html` - Main portfolio page
- `styles.css` - Responsive visual design
- `script.js` - Header and footer polish
- `assets/profile.png` - Profile image
- `assets/Mubashir-Hassan-Resume.pdf` - Downloadable resume

Project screenshots can be added under `assets/projects/` and wired into the project cards in `index.html`.
